# DriveElite 🚗

Site de venda de veículos desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Listagem de veículos em cards com foto, ano e preço
- Filtro por faixa de preço (popular, médio e luxo)
- Busca por nome do modelo
- Modal com detalhes do veículo
- Formulário de contato com preenchimento automático ao clicar em "Tenho Interesse"

## Tecnologias

- HTML
- CSS
- JavaScript

## Como usar

Basta abrir o arquivo `index.html` no navegador — não precisa instalar nada.

## Autor

Desenvolvido como projeto prático de front-end.

### Deploy
- Site publicado via Netlify sem necessidade de cartão de crédito
- Hospedagem gratuita com domínio gerado automaticamente (`seusite.netlify.app`)
- Deploy feito por arraste de pasta, sem necessidade de configuração
- HTTPS gerado automaticamente pelo Netlify
